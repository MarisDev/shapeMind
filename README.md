# ShapeMind

> Seu parceiro na evolução.

## 📖 Sobre o projeto

O ShapeMind é um aplicativo mobile em desenvolvimento para auxiliar pessoas que desejam construir uma rotina mais saudável, seja com foco em emagrecimento, manutenção do peso ou hipertrofia.

O objetivo é oferecer uma experiência simples, intuitiva e motivadora, permitindo que o usuário acompanhe sua evolução diária sem julgamentos.

Este projeto também faz parte da minha jornada de aprendizado em Engenharia de Software, sendo desenvolvido do zero para colocar em prática conceitos de desenvolvimento mobile, backend, banco de dados, arquitetura de software e inteligência artificial.

---

## 🎯 Objetivo

Desenvolver um aplicativo capaz de acompanhar a evolução física e alimentar do usuário, incentivando hábitos saudáveis por meio de acompanhamento contínuo e, futuramente, recursos de Inteligência Artificial.
